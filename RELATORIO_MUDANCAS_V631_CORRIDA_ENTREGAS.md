# Relatório v6.31.0 — Corrida e Entregas

Esta versão organiza a nomenclatura operacional do app do entregador:

- `Corrida` = missão do motoboy.
- `Entrega` = cada pedido/cliente dentro da corrida.
- `Parada` = ponto da sequência: loja, cliente 1, cliente 2.
- `Pedido`/`rotaId` = referência interna, não título principal.

## Ajustes aplicados

1. Removido destaque visual de `Rota #...` no cabeçalho principal.
2. Criados labels derivados: `1 entrega`, `2 entregas`, `Corrida agrupada`.
3. Card ativo da Home agora mostra quantidade de entregas e etapa.
4. Tela de Corridas agora mostra `Paradas da corrida`.
5. Mapa foi mantido, mas renomeado para `Mapa da corrida`.
6. Suporte/ocorrência/preferências usam linguagem de corrida.
7. Versão Android atualizada para `6.33.0-operacao-completa-tela-cheia`.
